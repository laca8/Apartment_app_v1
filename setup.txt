⚙️ Project Setup
1. Create .env file in backend
# MongoDB Connection
MONGO_URI=mongodb://mongo:27017/apartments_app

# Server Configuration
PORT=5000
NODE_ENV=development

2. Create .env file in frontend

NEXT_PUBLIC_API_URL=http://localhost:5000/api/v


3. cd docker
docker-compose up -d

4. go to 

http://localhost:3000
